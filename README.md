# 项目启动

npm i & npm run start

# 说明
* 项目适合刚开始使用ant-desgin-pro开发的小伙伴，里面权限管理，组件调用，自定义组件，状态管理，mock等等都使用到了，是一个小而全的项目。
* 项目使用mock假数据，api文档放在根目录api-doc文件夹下
* 后面如果二期开发，会继续更新这个项目
